About Python

Python is an interpreted, object-oriented, high-level programming language for dynamic data types. Python was invented by Guido van Rossum at the end of 1989, and the first public release was released in 1991. Like Perl, Python source code also follows the GPL (GNU General Public License) agreement. Python can also be used for the development of desktop windows applications, but the most important aspect is user big data mining.

Python 与 PyQt5

Python 是一种解释型、面向对象、动态数据类型的高级程序设计语言。 Python 由 Guido van Rossum 于 1989 年底发明，第一个公开发行版发行于 1991 年。 像 Perl 语言一样, Python 源代码同样遵循 GPL(GNU General Public License) 协议。 Python 一样也可以用桌面视窗应用的开发，不过更主要的是用户大数据挖掘这方面。
PyQt5 是 Python 的一个第三方库，用于开发可视化桌面应用程序用的， 是从 Qt 移植过来的，可以以纯代码的形式开发，也可以通过 qt-tools 提供的 可视化设计器进行开发，使用设计器时，遵循所见即所得的原则布置控件的位置和布局。 PyQt5 除了用于开发可视化 Python 应用，还可以进一步结合 qtpadas 进行数据可视化， 生成和制作美观的图表。

About PyQt5

PyQt5 is a third-party library for Python. It is used to develop visual desktop applications. It is ported from Qt. It can be developed in the form of pure code or through the visual designer provided by qt-tools. Use the designer Layout, follow the WYSIWYG principle to arrange the position and layout of the control. In addition to developing visual Python applications, PyQt5 can be further combined with qtpadas for data visualization, generating and making beautiful charts.
